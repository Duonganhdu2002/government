# Express PostgreSQL Redis API
```bash
Dự án backend sử dụng Express.js, PostgreSQL và Redis.
```

# Cài đặt 
```bash
git clone <git@github.com:Duonganhdu2002/government.git>
cd government
npm install
```

# Cấu hình
```bash
Tạo file .env và copy nội dung từ .env.example
```

# Chạy server
```bash
npm run dev
```
