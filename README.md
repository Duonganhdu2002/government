# Express PostgreSQL Redis API
```bash
Dự án backend sử dụng Express.js, PostgreSQL và Redis.
```

# Cài đặt 
```bash
git clone <git@github.com:Duonganhdu2002/government.git>
cd government
npm install
```

# Cấu hình
```bash
Tạo file .env và copy nội dung từ .env.example
```

# Chạy server
```bash
npm run dev
```

# Chạy Admin 
```bash
cd admin
npm install
npm run dev
```

# Chạy web user
```bash
cd web
npm install
npm run dev
```
# Chạy mobile
```bash
cd mobile
pub get
flutter run
```
